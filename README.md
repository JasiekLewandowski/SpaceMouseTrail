# OrderApp

OrderApp is a simple web application for customer service in a bar.

# Features

- Add new product
- Get list of avaible products
- Place an order
- Get list of placed orders
- Generate receipt (pay)

# Technologies

OderApp is created with:

* Spring Boot 2.4.1
* Maven
* H2 Database Engine
* Lombok
* Junit 5
* Mockito
